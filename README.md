# CaregiverSurveyApp
CaregiverSurveyApp is a tool to streamline the collection of intertemporal choice data for adults remarking on this preferences for treatment gains. It is built upon Cocosharp and Xamarin Forms to develop an interactive interface that will function across Windows, Android and iOS. It requires the REDCap back-end to successfully record user data.

Features include:
  - Native views in both iOS and Android
  - Dynamically added text
  - Teaching trials to minimize unsystematic responding
  - Store data in HIPPA-compliant REDCap server
